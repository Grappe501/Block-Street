# Calendar Event Report Protocol

1. After event end (or when marked completed), required report items become due within 48 hours.
2. Organizers capture metrics and complete follow-up actions on the event report checklist.
3. Submitting an item records soft-beta status only — no automated approval or durable persistence.
4. The `follow_up` readiness dimension reflects incomplete required items as **blocked** for completed/past events.

No automated outreach or CRM sync in Wave 3C.
