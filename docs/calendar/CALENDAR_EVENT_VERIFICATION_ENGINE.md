# Calendar Event Verification Engine

Soft-beta venue, campus, legal, and accessibility verification checklist — separate from calendar approval.

## Categories

| Category | Examples |
|----------|----------|
| venue | Layout and access confirmation |
| campus | Campus permission |
| legal | Compliance review |
| accessibility | Accessibility review |

## Template seed

`ensureVerificationFromEvent` seeds items when `verification` is a template readiness dimension or for public in-person events.

## Authority

Verification items carry `softBeta: true` and `durableAuthority: false` while Gate A is open.
