# Calendar Event Candidate Request Engine

Soft-beta Kelly attendance request workflow — request ≠ confirmation.

## Categories

| Category | Purpose |
|----------|---------|
| request | Submit request and define role scope |
| briefing | Briefing package preparation |
| travel | Travel and arrival review |
| communication | Public language approval |
| confirmation | Record confirmation separately from request |

## Template seed

`ensureCandidateRequestFromEvent` seeds from template `candidateRequest` defaults when Kelly is requested or candidate is a readiness dimension.

## Authority

All items carry `softBeta: true` and `durableAuthority: false` while Gate A is open.
