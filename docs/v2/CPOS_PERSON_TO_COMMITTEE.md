# CPOS Person-to-Committee 1.0

**Program ID:** `CPOS-PERSON-TO-COMMITTEE-1.0`

> Turn a newly recruited person into an active, connected committee member without requiring a human operator to explain every step.

## Canonical route

Volunteer-facing home: **`/home`**

## Build waves

| Wave | Deliverable | Status |
|------|-------------|--------|
| 1 | `/home` shell | In progress |
| 2 | Onboarding state machine | Planned |
| 3 | Outreach queue `/command/outreach` | Shell live |
| 4 | Committee bulletin board | Planned |
| 5 | Placement handoff | Planned |
| 6 | Certification journey | Planned |

## Onboarding states

invited → registered → profile_started → profile_complete → contacted → onboarding → placement_recommended → placed → activated

## Responsibility boundary

Outreach owns recruit through placement recommendation. Volunteer Manager owns final placement and training. Social Media owns public publishing only.
