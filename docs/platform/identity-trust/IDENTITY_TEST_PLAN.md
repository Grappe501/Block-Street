# Identity Test Plan

**AC-ITF-001 · Phase 10.6A**

## 14-Point Acceptance Test

1. No public registration exists
2. Every account enters through invitation
3. Every invitation has a permanent sponsor
4. Every sponsor permanently linked to invitations
5. Every user has a Global Human ID
6. Identity history is immutable
7. Multiple institutions share one human without sharing permissions
8. Verification history remains private
9. Public identity shows approved names/aliases only
10. Identity review process functions
11. Appeals process functions
12. Duplicate identities detected and investigated
13. AI recommends but cannot approve/deny identities
14. All events recorded in immutable audit trail
