# Trust Lifecycle

**ITL-TRU-001**

## Stages

Pending → Sponsored → Verified → Trusted → Institution Leader → Federation Trusted

## Rules

- Automatic promotion when independent verifications and participation thresholds met
- Manual review for edge cases
- Dormancy detection (configurable days)
- Recovery and appeals via governance
- Trust demotion with audit record

## APIs

- `GET /api/v1/identity-trust/trust`
