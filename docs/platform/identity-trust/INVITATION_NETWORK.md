# Invitation Network

**ITL-INV-001**

## One Way In

No public registration page. Invitation only.

## Permanent Record

Every invitation records: creator, institution, organization, date, reason, intended role, expiration, acceptance.

## Outputs

Invitation tree, invitation graph, invitation analytics.

## APIs

- `GET/POST /api/v1/identity-trust/invitations`
- `POST /api/v1/identity-trust/register`
- `GET /api/v1/identity-trust/invite-tree`
