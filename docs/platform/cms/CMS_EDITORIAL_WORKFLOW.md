# CMS Editorial Workflow

## Workflow Types

- **Simple:** draft → publish
- **Standard:** draft → editor review → approval → publish
- **High-risk:** draft → SME → legal → executive → scheduled → publish

## Editorial Permissions

`content.view` · `content.create` · `content.edit_own` · `content.edit_all` · `content.submit_review` · `content.review` · `content.approve` · `content.publish` · `content.schedule` · `content.archive` · `media.upload`
