# Community Network Model

**System ID:** REL-001

## Hierarchy

```
People → Teams → Organizations → Communities → Counties → Regions → Federation
```

## Community Connectors

People who naturally bridge communities:

- Mentors
- Organizers
- Coalition builders
- Trainers
- Multi-organization volunteers

Identified via `identifyConnectors()` when a person connects 2+ organizations or counties.

## Community Bridges

- Shared projects
- Coalition relationships
- Regional connectors

## Isolation Detection

Early intervention targets:

- Teams with limited interaction
- New volunteers without mentors
- Organizations disconnected from partnerships
- Counties lacking collaboration
