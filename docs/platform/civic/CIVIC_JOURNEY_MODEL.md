# Civic Journey Model

**System ID:** CIV-001

## Lifecycle

Discover → Join → Learn → Participate → Return → Contribute → Lead → Mentor → Build Community

## Stages

Interested → Participant → Regular Contributor → Reliable Volunteer → Leader → Community Builder → Mentor → Institution Builder → Civic Champion

Movement depends on meaningful activity—not tenure. Each participant maintains a lifelong civic timeline.

**API:** `GET /api/v1/civic/timeline`
