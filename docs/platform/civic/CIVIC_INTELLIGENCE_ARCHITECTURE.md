# Civic Intelligence Architecture

**System ID:** CGI-001 · Phase 10

```
Institution → People → Participation → Relationships → Communities → Civic Outcomes → Societal Impact
```

Phase 10 consumes data from identity, organizations, missions, training, analytics, AI, federation, support, calendar, and communications—no duplicate data.

**Governing principle:** The purpose of the platform is not to create activity. The purpose is to create stronger communities.
