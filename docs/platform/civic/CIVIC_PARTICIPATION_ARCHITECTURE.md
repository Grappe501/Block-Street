# Civic Participation Architecture

**System ID:** CIV-001 · Build 10.1

```
Institution → People → Participation → Relationships → Communities → Civic Outcomes → Societal Impact
```

The Civic Participation Engine transforms software activity into measurable civic participation. It consumes identity, missions, training, operations, and federation data without duplication.

**Engine:** `src/lib/civic/` · **Data:** `data/civic/`

**Governing principle:** Participation is measured by meaningful contribution—not by login counts.
