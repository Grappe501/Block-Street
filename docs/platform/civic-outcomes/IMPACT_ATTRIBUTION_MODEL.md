# Impact Attribution Model

**System ID:** OUT-001

## Attribution Levels

1. **Correlation** — observed together, no causal claim
2. **Contribution** — program plausibly contributed
3. **Likely Attribution** — strong evidence of contribution
4. **Direct Attribution** — very high confidence with baseline + verified evidence

Confidence always displayed. Never claim more certainty than evidence supports.

API: `GET /api/v1/civic-outcomes/attribution?outcome_id=`
