# Public Human Identity Policy

Every active user must display the name by which they are genuinely known to other humans.

## Permitted

- Everyday public names (`Steve Grappe`)
- Professional or community names
- Approved established public aliases
- Religious or clerical names when genuinely used

## Prohibited

- Disposable screen names (`Patriot1776`, `Volunteer42`)
- Organization names posing as individuals
- Political slogans or impersonation
- Concealed non-human identities

## Legal name

Optional for ordinary platform use. When collected for lawful purpose:

- Separately protected
- Access-restricted
- Never replaces public name by default

## Name changes

Public identity versions are append-only. Prior names remain in governed history.

Validation: `validatePublicHumanName()` in `src/lib/identity-trust/wave1/engine.ts`
