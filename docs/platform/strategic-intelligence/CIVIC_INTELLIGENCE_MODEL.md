# Civic Intelligence Model

**System ID:** INT-002

## StrategicInsight

| Field | Description |
|-------|-------------|
| category | opportunity, risk, trend, capacity, leadership, etc. |
| priority | low, medium, high, critical |
| confidence | low, moderate, high, very_high |
| supporting_evidence | Transparent evidence list |
| explainable_reasoning | Why this insight was generated |
| alternatives | Other options leaders may consider |

Every insight is explainable, evidence-based, and optional.
