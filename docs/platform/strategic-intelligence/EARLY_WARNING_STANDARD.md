# Early Warning Standard

**System ID:** INT-002

## Warning Levels

| Level | Use |
|-------|-----|
| Informational | Emerging pattern |
| Watch | Monitor closely |
| Concern | Intervention recommended |
| High Risk | Urgent attention |
| Critical | Immediate action needed |

Every warning includes supporting evidence and explainable reason.

API: `GET /api/v1/strategic-intelligence/warnings?refresh=true`
