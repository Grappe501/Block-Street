# Community Resilience Model

**System ID:** CHD-001

## Resilience States

| State | Threshold |
|-------|-----------|
| Fragile | < 30 |
| Developing | 30–49 |
| Stable | 50–67 |
| Resilient | 68–81 |
| Highly Resilient | 82–94 |
| Model Community | ≥ 95 |

## Factors

- Leadership redundancy
- Volunteer redundancy
- Organizational diversity
- Partnership diversity
- Collaboration continuity
- Knowledge transfer via mentorship

API: `GET /api/v1/community-health/resilience`
