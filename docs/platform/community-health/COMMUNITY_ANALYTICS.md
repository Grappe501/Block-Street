# Community Analytics

**System ID:** CHD-001

## Analytics Surfaces

| Endpoint | Purpose |
|----------|---------|
| `/analytics` | County executive dashboard |
| `/ecosystem` | Organizational ecosystem health |
| `/risks` | Early warning detection |
| `/opportunities` | Civic opportunity mapping |
| `/geographic` | Permission-aware health maps |
| `/trends` | Longitudinal community trends |
| `/benchmark` | Anonymous peer comparison |

## Risk Types

Leadership shortage, volunteer burnout, organizational isolation, weak collaboration, declining participation, mentor shortage, aging leadership, succession gap.
