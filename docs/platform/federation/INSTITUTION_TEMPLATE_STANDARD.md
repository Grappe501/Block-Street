# Institution Template Standard

**System ID:** FED-001

Templates include organizational structure, roles, permissions, training, missions, dashboards, playbooks, workflows, and automation. Templates exclude personal data, private communications, authentication secrets, and confidential records.

## Template Package Fields

- `id`, `name`, `description`, `version`
- `supported_modules`, `organization_model`, `training_model`
- `playbooks`, `workflows`, `permissions`, `default_dashboards`
- `recommended_settings`, `status`

## Template Types

Campus, county, state, coalition, university, nonprofit, campaign, advocacy, community, educational institution.
