# Task Domain Services

**Protocol:** CAE-11.2-W3

| Service | Responsibility |
|---------|----------------|
| TaskLifecycleService | draft → assigned → active → completed |
| TaskAssignmentService | AssignTask |
| TaskDependencyService | Dependency blocking |
| TaskEvidenceService | AttachEvidence linkage |

Orphan tasks prohibited (OBJ-V-021). Task lifecycle cannot exceed Mission lifecycle.
