# Objective Taxonomy

**Build:** 11.2 · **Wave:** W1 · **Canonical source:** `data/phase-11/objective_types.json`

Thirteen primary types. One primary type per Objective. Types influence recommendations but **do not change governance**.

| Type | Summary |
|------|---------|
| Strategic | Long-horizon institutional direction |
| Operational | Day-to-day execution outcomes |
| Community | Community-facing change |
| Legislative | Policy and civic outcomes |
| Educational | Learning and certification outcomes |
| Campaign | Time-bounded mobilization targets |
| Technology | Platform and data outcomes |
| Financial | Funding and sustainability |
| Volunteer | Recruitment and retention |
| Membership | Growth and engagement |
| Training | Skill development |
| Research | Evidence and findings |
| Emergency | Urgent short-horizon outcomes |
