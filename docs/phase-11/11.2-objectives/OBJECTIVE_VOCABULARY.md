# Objective Vocabulary

**Build:** 11.2 · **Wave:** W1 · **Canonical source:** `data/phase-11/objective_vocabulary.json`

| Term | Definition | Prohibited uses |
|------|------------|-----------------|
| Objective | Future state the institution intends to create | task, meeting, checklist |
| Key Result | Measurable indicator of Objective progress | purpose statement only |
| Workstream | Major execution line under one primary Objective | orphan container |
| Mission | Smallest independently meaningful work unit | initiative synonym |
| Milestone | Meaningful checkpoint | deliverable synonym |
| Deliverable | Tangible Mission output | milestone synonym |
| Task | Executable action within a Mission | objective, orphan work |
| Orphan work | Execution without Objective ancestry | allowed pattern |
