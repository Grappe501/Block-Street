# Objective Traceability Standard

**Build:** 11.2 · **Wave:** W1

## Upward trace (required)

Every Task must resolve:

```text
Task → Mission → Workstream → Objective → Initiative → Institution
```

## Cross-cutting traces

| Artifact | Must trace to |
|----------|---------------|
| Calendar event | Objective |
| Resource allocation | Objective |
| Budget item | Objective |
| Volunteer assignment | Objective |
| Communication campaign | Objective |
| Automation | Objective |

## Permanent history

Completed work retains traceability permanently. Silent deletion of ancestry is prohibited.

## Platform question

> Why does this Task exist?

The platform must always produce a human-readable answer with evidence links.
