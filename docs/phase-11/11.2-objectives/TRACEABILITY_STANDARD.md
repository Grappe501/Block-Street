# Traceability Standard

**Protocol:** CAE-11.2-W2 · **Implementation:** `src/lib/civic-action/builds/11.2/traceability.ts`

## Upward chain

Task → Mission → Workstream → Objective → Initiative → Institution

## Platform question

> Why does this Task exist?

Must answer instantly with human-readable chain and evidence links.

## Reverse lookup

Given any Objective, list all descendant Missions and Tasks.

## Cross-cutting

Calendar, resources, budget, communications, automations trace to Objective.
