# Communication Workbench Protocol — CAE-11.7-W4

**Subsystem:** COM-UX-001

The Collaboration Workbench is the Human operating system for institutional communications. Role-aware navigation adapts for executive, operational owner, mission lead, contributor, volunteer, and viewer roles.

## Governed actions only

Lifecycle changes use explicit action buttons. **No status dropdown.**

## Shell navigation

My Communications · Mission Conversations · Meetings · Documents · Decisions · Knowledge · Notifications

## Application service

`CommunicationApplicationService` wraps `communicationsDomainService` and provides read helpers for assemblers.
