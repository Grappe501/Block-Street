## Identity Trust Integration

Every initiative action resolves:

```text
Global Human ID · Institution Membership · Active Institution Context
Initiative Role · Permission · Request ID · Correlation ID
```

Cross-institution participation only through: coalition agreement, shared program membership, approved partner role, temporary assignment, cross-institution invitation.

Participation does not create general membership in host institution.

## Visibility Does Not Equal Access

See constitution. Enforcement via permission model in 11.1-W3.

## Public Participation

Public feedback, registration, and applications do not automatically create full platform accounts. Account creation follows Identity Trust Layer.
