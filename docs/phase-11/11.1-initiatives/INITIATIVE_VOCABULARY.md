# Initiative Vocabulary

**Build:** 11.1 · **Wave:** W1 · **Canonical source:** [01_CONSTITUTION.md](01_CONSTITUTION.md)

Machine-readable registry: `data/phase-11/initiative_vocabulary.json`

## Core Terms

| Term | Definition |
|------|------------|
| Initiative | Root container for coordinated action |
| Program | Recurring or continuing initiative delivering ongoing value |
| Campaign | Time-bounded effort pursuing a defined target |
| Project | Bounded effort producing a defined deliverable |
| Operation | Coordinated execution effort, often time-sensitive |
| Pilot | Limited initiative designed to test and learn |
| Charter | Governed statement of purpose, ownership, scope, and authority |
| Executive Owner | Human accountable for institutional legitimacy and strategic outcome |
| Operational Owner | Human accountable for execution |
| Backup Owner | Human authorized when primary operational owner unavailable |
| Sponsoring Authority | Human or body authorizing the initiative |
| Scope | Boundary of authority, work, geography, Humans, data, resources, time |
| Lifecycle | Governed sequence from concept through archive |
| Dependency | Required relationship to another initiative, authority, resource, or condition |
| Portfolio | Institution's complete set of initiatives |
| Closeout | Governed process of ending execution and preserving results |
| Successor Initiative | New initiative continuing or replacing prior work |
| Workstream | Major line of work inside an initiative |
| Mission | Executable unit of action inside a workstream |
| Owner Required | Exception state when no eligible operational owner exists |

## Prohibited Uses

Terms must not be redefined in downstream builds. Campaign, project, and program are **types** or **contents** of initiatives, not parallel root governance objects.
