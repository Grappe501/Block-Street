# Initiative Dependency Engine

**Build:** 11.1 · **Wave:** W3

`services/dependency-graph.ts`

## Capabilities

- Validate new dependencies (no self-reference, no duplicates)
- Detect circular blocking cycles (`CIRCULAR_DEPENDENCY_DETECTED`)
- Evaluate readiness for activation
- Blocking vs advisory dependencies

## Waiver

Waivers require authority, rationale, duration, and audit. Constitutional requirements cannot be waived.

## Supersession

Successor relationships preserve both initiative histories.
