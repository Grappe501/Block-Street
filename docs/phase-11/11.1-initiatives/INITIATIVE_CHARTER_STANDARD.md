# Initiative Charter Standard

**Build:** 11.1 · **Wave:** W1 · **Canonical source:** [01_CONSTITUTION.md](01_CONSTITUTION.md)

## Required Charter Sections

```text
Initiative Name · Public Name · Initiative Type · Governing Institution
Sponsoring Authority · Executive Owner · Operational Owner · Backup Owner
Problem or Opportunity · Purpose · Institutional Alignment · Intended Change
Population · Geographic Scope · Functional Scope · In Scope · Out of Scope
Start Date · Target End Date · Review Rhythm · Approval Requirements
Visibility · Risks · Dependencies · Closeout Basis
```

Objectives and detailed metrics are added through Build 11.2.

## Charter Status

```text
incomplete → draft → ready_for_review → approved → active_version → superseded → archived
```

## Naming Standard

Names must be specific, recognizable, distinguishable, human-readable, and stable for history.

**Good:** `2027 Arkansas Youth Civic Academy`  
**Avoid:** `Project 1`, `New Initiative`, `Test`, `Campaign`

## Public vs Internal Name

Both names link to one initiative record. Example:

- Internal: Identity Trust Layer Controlled Launch
- Public: Our Verified Community

## Versioning Triggers

New charter version required for: purpose change, type change, major scope change, governing institution change, executive ownership change, material timeline change, risk class change, target population change, public status change.

Minor editorial corrections remain in history without full re-approval (institution policy defines threshold — POL-007).

Entity implementation: **11.1-W2** (`InitiativeCharter`).
