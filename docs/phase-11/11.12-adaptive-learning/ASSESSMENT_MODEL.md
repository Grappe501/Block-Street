# Assessment Model

**Protocol:** CAE-11.12-W2

## Assessment

Defines passing score, max attempts, and optional competency/course binding.

## AssessmentAttempt

Human attempt with lifecycle: `in_progress → submitted → graded → archived`.

## AssessmentResult

Graded outcome with human grader authority. `passed` boolean and numeric `score`.

## Constitutional rule

Assessment proves demonstrated capability; it does not substitute for evidence-backed claims or human certification authority.
