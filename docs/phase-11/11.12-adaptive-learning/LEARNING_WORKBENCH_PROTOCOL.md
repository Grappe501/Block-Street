# Learning Workbench Protocol

**Protocol ID:** CAE-11.12-W4 · **Contract:** KNW-UX-001

## Universal layout

Top navigation, left nav (Mission, Knowledge, Learning, Competencies, Certifications), main workspace, context panel.

## Role-aware navigation

Learners see a focused nav; stewards and administrators see research and governance sections.

## Governed actions

UI exposes command endpoints — no status dropdowns. Mutations route through Wave 3 domain commands.

## Version awareness

Historical knowledge displays: "You are viewing a historical version. Current version available."
