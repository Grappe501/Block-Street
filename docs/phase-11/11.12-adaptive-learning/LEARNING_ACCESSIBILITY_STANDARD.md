# Learning Accessibility Standard

**Protocol ID:** CAE-11.12-W4

- Keyboard navigation for workbench shell nav
- `aria-label` on navigation and context panel
- `aria-current="page"` on active nav item
- Historical content uses `role="status"` banner
- Reduced motion and high contrast supported via design tokens
- Scalable text via responsive typography
- Analytics measure workflow friction — not Human worth
