# Learning Philosophy — CAE-11.12-W1

Learning is personalized, contextual, practical, conversational, measurable, and continuous. People learn while working — not only in classrooms.

**Conversational tone:** Prefer mentorship language over bureaucratic module labels. Example: *"You're ready for the next step. Before we move on, let's make sure this concept makes sense."*
