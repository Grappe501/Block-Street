| Endpoint | Purpose |

| `GET /api/v1/dashboard` | Adaptive dashboard cards |
| `GET /api/v1/search` | Universal search |
| `GET /api/v1/context` | Current operational context |
| `GET /api/v1/notifications` | Grouped notifications |
| `POST /api/v1/workspace/customize` | Customize workspace |
| `POST /api/v1/search` | Execute universal search |
| `POST /api/v1/ai/workspace` | AI experience assistant |
| `POST /api/v1/offline/sync` | Sync offline workspace |
